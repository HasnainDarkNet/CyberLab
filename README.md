## CyberLab 🔒

**By HasnainDarkNet**

CyberLab is an **interactive educational tool** designed to teach students and beginners the risks of unsafe file downloads. It simulates real-world scenarios showing how downloading unknown files can compromise your system, while providing a **safe environment** to learn **ethical hacking** and **cyber security awareness**.

---

## 🛠 Features

- Terminal-controlled file selection for live demonstration  
- Hacker-style web interface with dark mode aesthetics  
- Safe download simulation for educational purposes  
- Works on **Windows, Mac, and Linux**  
- Ideal for students, beginners, and cyber security enthusiasts  

---

## 🎨 User Interface

- Dark terminal-inspired web page  
- OS sections: Windows, Mac, Kali Linux  
- Interactive download buttons linked to selected files  
- Step-by-step safe download instructions
